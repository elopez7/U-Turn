The Code for U-TURN

                        
                            document.getElementById("btnSubmit").addEventListener("click", getStringFromPage);

                            function getStringFromPage(){
                                document.getElementById("alert").classList.add("invisible");
                            
                                let inputString = document.getElementById("userString").value;
                                let reversedString = reverseString(inputString);
                                displayReversedString(reversedString);
                            }

                            function reverseString(stringToReverse){
                                let reversedString = [];
                            
                                for(index = (stringToReverse.length - 1); index >= 0; index--){
                                    reversedString += stringToReverse[index];
                                }
                            
                                return reversedString.toString();
                            }

                            function displayReversedString(stringToDisplay){
                            
                                document.getElementById("msg").innerHTML = `Your U-Turned string is: ${stringToDisplay}`;
                                document.getElementById("alert").classList.remove("invisible");
                            }

The Code is structured in an even listener and three functions.

First Line

The first line wires up our submit button to a click event listener that calls getStringFromPage().

getStringFromPage()

Ensures that our output section in the page is initially invisible, then takes the user input and passes it to reverseString(), the reversed string will be assigned to the reversedString variable which is then passed to displayReversedString() to display the results to the screen..

reverseString()

Takes the input string as a parameter, then using a for loop the reversedString list is assigned the characters of stringToReverse from last to first, thus creating a reversed string which is then returned by the function.

displayReversedString()

Takes the results from reverseString() displays it to the user by making the output area visible.
